Troubleshooting Guide

  • 400 Error: Missing or invalid Kowri credentials in Supabase secrets
  • Connection Error: Check if KOWRI_BASE_URL is correct
  • Invalid phone: Use valid MTN Ghana number (024...)
  • Check browser console: F12 → Console tab for detailed errors
  • Webhook not firing: Verify KOWRI_WEBHOOK_URL matches your domain
